# Solar Sentinel Data Retention Policy

This document outlines the formal storage rules and timelines for artifacts generated by the Solar-Flare-Forecasting pipeline.

## 1. Raw Data
- **Raw FITS files**: Retain for **90 days**, then delete. (Storage: ~5GB/month)
- **Processed parquet cache**: Retain for **365 days**.

## 2. Master Catalogue
- **`master_catalogue.csv`**: **PERMANENT**. This is the primary PS15 deliverable and must not be deleted.

## 3. MLflow Models & Experiments
- **Active / Promoted Models**: Retain for **365 days**.
- **Rejected Runs (tag: `promotion_status=rejected_*`)**: Retain for **30 days**.

## 4. Observability
- **Grafana Metrics (Prometheus)**: Retain for **90 days** (handled via Prometheus native settings).
- **Logs (`logs/`)**: Retain for **30 days** and rotated daily. Hard limit of **1GB total** space.

## Automation
The data retention policy is automated via `scripts/cleanup_old_data.py`, which is scheduled to run daily at 02:00 via a cron job on the worker instances.
